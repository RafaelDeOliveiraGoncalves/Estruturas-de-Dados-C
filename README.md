# Estruturas de Dados em C

Este repositório contém implementações robustas e modulares de diversas estruturas de dados clássicas, desenvolvidas em linguagem C para fins de estudo e aplicação prática.

Observação: As estruturas de dados apresentadas foram implementadas utilizando o tipo int. Embora a linguagem C permita o uso de tipos genéricos (ponteiros void) optou-se por não utilizá-los, pois, é bastante complexo. Caso deseje utilizar outros tipos de dados, a lógica permanece a mesma, sendo necessário apenas ajustar as declarações de tipo e as funções de manipulação.

## Estruturas Implementadas

### 1. Lineares:

- Array Dinâmico: Vetor com redimensionamento automático de capacidade.

- Deque Dinâmico: Fila de duas extremidades com buffer circular e redimensionamento.

- Filas (Queue): Implementações em Deque (circular), encadeada e com prioridade (Heap).

- Pilhas (Stack): Versões dinâmicas e encadeadas.

- Lista Encadeada: Implementação dinâmica de lista com nós.

### 2. Não Lineares:

- Árvores: Árvore Binária de Busca (BST) e Árvore AVL auto-balanceada.
